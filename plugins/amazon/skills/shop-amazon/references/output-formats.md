# Output Formats

```markdown
## Amazon Shortlist - [Category]

### 1. [Product] - $XX.XX (verified)
**ASIN**: B0XXXXXXXX
**Rating**: X.X/5 (X,XXX reviews)
**Amazon**: https://www.amazon.com/dp/B0XXXXXXXX

```

Use `(verified)` only after checking title, price, rating, and review count on the product page.
